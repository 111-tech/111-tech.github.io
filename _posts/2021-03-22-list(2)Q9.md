# 문제 상황
KBO는 홈페이지에서 선수조회 서비스를 제공하고 있습니다. 새로운 팀이 창단되면 이것을 기존의 선수명단에 추가됩니다. 선수명단에서 구본혁의 등번호와 포지션을 알아보는 프로그램을 작성하시오. 단, 인덱스는 사용자로부터 입력받도록 한다.

# 문제 해결 프로그래밍

~~~python

print("[0]강민", "[1]고우석", "[2]구본혁", "[3]김대유", "[4]김대현\n")
print("원하시는 선수의 인덱스 번호를 입력한 후 엔터 키를 눌러주세요.\n")
user_input = int(input("원하는 선수의 인덱스 번호 입력: "))
print(playerinfo[user_input])

# Q9 - 2
back = [105, 19, 25, 69, 45]

  print("생년월일:", bir[user_input])
  print("체격:", body[user_input])
~~~
